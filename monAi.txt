💁‍♂️ MONAI is a PyTorch-based, open-source framework for deep learning in healthcare imaging, part of the PyTorch Ecosystem.

👏 Features
---------------
📌 The codebase is currently under active development.
📌 flexible preprocessing for multi-dimensional medical imaging data;
📌 compositional and portable APIs for ease of integration into existing workflows;
📌 domain-specific implementations for networks, losses, evaluation metrics, and more;
📌 customizable design for varying user expertise;
📌 multi-GPU data parallelism support.
